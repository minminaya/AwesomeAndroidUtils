# SizeUtils




常用的尺寸相关操作的工具类整理


# 基本方法

| 方法                | 作用                                                         |
| :------------------ | :----------------------------------------------------------- |
| ```dp2px```         | dp转px                                                       |
| ```px2dp```         | px转dp                                                       |
| ```calculatPoint``` | 知道一个线段，它的定点，线段旋转角度(线段起始角度为Android坐标系90°)求旋转后的终点坐标 |
| getScreenParams     | 获取DisplayMetrics，通过DisplayMetrics对象可以获取屏幕宽高，密度等 |
